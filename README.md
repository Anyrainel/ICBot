# ICBot

Illusion Connect bot created using Airtest.

It supports:
- Arena
- Ares Battlefield (naive)
- Partner Date (partial)
- Dream Expedition
- Exploration (partial)
- Daily Trial
- Buy Stamina
- Guild Contribution
- Daily Quest

## How to use

[Demo](https://youtu.be/HWGXSp8aMB0)

[Setup Guide](https://youtu.be/tU9veFLHgHo)

## Set up

- Install [AirtestIDE](http://airtest.netease.com/)
- Set up your Android emulator with IC app. (Default setting uses LDPlayer.)
- Enable ADB Debug in emulator.
- Copy `settings.default.jsonc` to `settings.jsonc` and edit.
- Open IC then run `python ICBot.py` and enjoy!
